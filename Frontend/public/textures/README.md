# Normal Map Texture

To make the planets look realistic, download the 1k normal map from NASA:

URL: https://svs.gsfc.nasa.gov/cgi-bin/details.cgi?aid=4720
File: Download the 1k normal map image.
Rename: Name the file `moon-normal.jpg`
Location: Place it in this `/public/textures/` directory.

The ProjectOrb.tsx component expects this texture to exist at `/textures/moon-normal.jpg`.
